# cnd-cfp-prompts

CloudNative Days (CND) の CfP（セッション公募）を、選考委員の視点でセルフレビューするための LLM 用プロンプト置き場です。

## 概要

このリポジトリには、CND のプログラム選考委員（PC）の知見をシミュレートしたプロンプトを格納しています。 単なる文章添削ではなく、「クラウドネイティブコミュニティが今求めている知見か」「参加者の課題を解決できるか」という、CND ならではの選考基準に照らしたフィードバックを得ることを目的としています。

## 収録プロンプト

review.md: 選考委員視点での総合レビュープロンプト

* コミュニティ適合性の判定
* 具体的改善ポイント（再現性、独自知見、中立性など）の指摘
* 挑戦を後押しする伴走型のフィードバック

## 使い方

1. review.md の中身をすべてコピーします。
2. ChatGPTやClaude、Geminiなどの LLM に貼り付けます。
3. LLM から「CfPのレビューを承ります...」という返答が来たら、作成中の CfP 草案（タイトル、概要、対象者）を入力してください。

## プロンプトの設計思想

CND の過去の採択傾向や、エンジニアが「聴いてよかった」と思えるセッションの特徴に基づき、以下の「レビューの5箇条」を評価軸に組み込んでいます。

* 「なぜ今、これなのか」の明確さ
* 「泥臭い現実」の開示（失敗談や組織的な壁）
* 再現性（聴講者が持ち帰れる教訓）
* タイトルの「誠実な引き」
* 抽象度と具体度のバランス

## 注意事項

* 本プロンプトは、ユーザーが自らの言葉で最高のプロポーザルを紡ぎ出すことを支援するために「ヒントと問いかけ」に留めるよう設計されています。
* 登壇への挑戦を尊重し、ポジティブな要素を必ず見つけるように設定されています。
